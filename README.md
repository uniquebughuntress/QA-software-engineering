## 🧠 Mein QA-Mindset

Ich teste nicht nur "Funktioniert es?" – ich frage:

- Was passiert bei unerwarteten Eingaben?
- Was passiert bei unterbrochenen Prozessen?
- Was passiert, wenn zwei User dasselbe gleichzeitig tun?
- Was passiert nach 1000 Datensätzen?
- Was passiert, wenn die Session abläuft?
